2006 - Web semántica

Emplea búsquedas en lenguaje natural y la minería de datos. Clasifica la información de manera más eficiente, a fin de devolver resultados más precisos a las solicitudes de búsqueda de los usuarios.

La Web Semántica es una extensión de la World Wide Web que añade significado a la información en internet para que las máquinas puedan entenderla mejor. Utiliza metadatos y ontologías para estructurar los datos y facilitar su interpretación. Esto permite una mayor interoperabilidad entre sistemas, facilita la búsqueda de información y abre nuevas oportunidades para aplicaciones inteligentes y personalizadas.

2006 - DBpedia

Se lanza DBpedia, una base de datos semántica extraída de Wikipedia. DBpedia es un proyecto para la extracción de datos de Wikipedia para proponer una versión Web semántica. Este proyecto es realizado por la Universidad de Leipzig, Universidad Libre de Berlín y la compañía OpenLink Software.

DBpedia es un proyecto colaborativo que extrae información estructurada de Wikipedia y la convierte en una base de conocimiento semántica en formato de datos enlazados. Proporciona acceso a una gran cantidad de datos estructurados que pueden ser consultados y utilizados por máquinas para enriquecer aplicaciones web y análisis de datos.

2009 - Bitcoin

El lanzamiento de Bitcoin marcó el inicio de la popularización de la tecnología blockchain, que es fundamental para muchos conceptos de Web 3.0. Bitcoin introdujo la idea de una moneda digital descentralizada y un sistema de registro inmutable.

Bitcoin es la primera criptomoneda descentralizada, creada en 2009 por una persona o grupo bajo el seudónimo de Satoshi Nakamoto. Su creación marcó el inicio de la era de las criptomonedas y la tecnología de la cadena de bloques (blockchain). El objetivo inicial de Bitcoin era proporcionar un sistema de efectivo electrónico peer-to-peer, que permitiera realizar transacciones financieras sin la necesidad de intermediarios como bancos o gobiernos. Desde entonces, Bitcoin ha experimentado una volatilidad significativa en su valor, atrayendo la atención de inversores, entusiastas de la tecnología y reguladores financieros en todo el mundo.

2007 - Knowledge Graph

Google lanza el proyecto Knowledge Graph para mejorar los resultados de búsqueda. Además de que permite a Google entender las conexiones entre diferentes entidades y ofrecer respuestas más directas.

Knowledge Graph es una base de datos de conocimiento desarrollada por Google para mejorar los resultados de búsqueda. Fue introducido como parte del esfuerzo de la empresa por comprender mejor el contexto de las consultas de búsqueda y proporcionar respuestas más relevantes y completas. El Knowledge Graph utiliza una estructura de grafo para conectar entidades y sus relaciones, permitiendo a Google entender la semántica detrás de las consultas de búsqueda y ofrecer información directamente en la página de resultados, en forma de paneles de conocimiento. Esto ayuda a los usuarios a encontrar respuestas rápidas y completas sin tener que hacer clic en múltiples enlaces.

2014 - JS Frameworks

Nacen para la facilidad de creación de aplicaciones webs más dinámicas. Son entornos de aplicaciones web escritos en JavaScript que nos ofrecen una colección de componentes y funcionalidades.

Los frameworks de JavaScript son conjuntos de herramientas predefinidas y bibliotecas de código que proporcionan estructuras y funcionalidades para facilitar el desarrollo de aplicaciones web. Estos frameworks permiten a los desarrolladores crear aplicaciones de manera más eficiente al proporcionar una base sólida y abstracciones que simplifican tareas comunes, como la manipulación del DOM, el enrutamiento, la gestión del estado y las solicitudes HTTP. Algunos ejemplos populares de frameworks de JavaScript incluyen Angular, React y Vue.js. Cada uno tiene sus propias características y enfoques, lo que permite a los desarrolladores elegir el que mejor se adapte a las necesidades de su proyecto. Estos frameworks han revolucionado el desarrollo web moderno al proporcionar herramientas poderosas para la creación de aplicaciones interactivas y receptivas.

2015 - Progressive Web

Las Progressive Web Apps (PWAs) comenzaron a ganar popularidad alrededor de 2015, cuando fueron introducidas por primera vez por Google. Sin embargo, el concepto de aplicaciones web progresivas ha evolucionado a lo largo del tiempo con el avance de las tecnologías web y móviles.

Las Progressive Web Apps (PWAs) son aplicaciones web que utilizan tecnologías web modernas para ofrecer una experiencia de usuario similar a la de una aplicación nativa en dispositivos móviles y de escritorio. Estas aplicaciones se caracterizan por ser confiables, rápidas y engaging. Utilizan características como el Service Worker para habilitar funciones offline y ofrecen notificaciones push para mejorar la participación del usuario. Además, las PWAs se pueden instalar en el dispositivo del usuario y tienen la capacidad de acceder a funciones nativas del dispositivo, como la cámara o el GPS. Su enfoque progresivo significa que pueden funcionar en cualquier navegador y dispositivo, lo que las convierte en una opción versátil para desarrolladores y usuarios.

Denota aplicaciones web con mayores experiencias. Las PWAs están diseñadas para ser rápidas, confiables y ofrecer una experiencia de usuario atractiva, incluso en condiciones de red .
